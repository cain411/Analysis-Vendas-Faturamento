	Análise de Vendas de Lojas de Roupas:
Este projeto busca realizar uma análise exploratória de dados de vendas a partir da base "Vendas.xlsx", com foco no faturamento por loja e por tipo de produto.


	Objetivo:
Analisar o desempenho de vendas de uma rede de lojas, identificando:
-Lojas com maior faturamento;
-Produtos com melhor desempenho de vendas;
-Padrões relevantes para tomada de decisão.


	Base de Dados:

Informações que estão na base:
-Loja;
-Produto;
-Quantidade vendida;
-Valor total de vendas.


	Tecnologias/bibliotecas Utilizadas:

-Rstudio;
-tidyverse;
-ggplot2;
-readxl;
-dplyr.


	Análises Realizadas:

Faturamento por Loja:
-A loja **Iguatemi Campinas** apresentou o maior faturamento do período analisado, destacando-se significativamente em relação às demais.


Faturamento por Produto:
-O produto **Bermuda Lisa** foi o campeão de vendas, sendo responsável pelo maior volume de faturamento, especialmente na loja Iguatemi Campinas.


	Conclusão:
-Existe uma forte concentração de faturamento na loja Iguatemi Campinas;
-A Bermuda lisa é o produto mais relevante em questão comercial;
-Os dados indicam oportunidades de estratégia focada em produtos líderes e lojas de alto desempenho.
